- **Server/Hosting**: Railway
- **ORM**: Prisma
- **Backend**: MySQL

## Features

- Client
  * See the Menu   
  * Edit the Order
  * Add Name
  * See the total and subtotal
- Admin 
  * Review the orders in real time
  * Mark as completed
  * Orders organized by status
